## DESCRIPTION: 
                This project is a simple c++ project without using oop. It involves using simple header files, data files and implementation files. This is a snake game where a snakes tail grows if it eats fruits and dies as soon as it touches a boundary.

## Features
            - Real-time keyboard input
            - Snake movement and tail growth
            - Fruit generation
            - Collision detection
            - High score saving
            - Settings management
            - Simple console graphics
            -Using binary file to store settings and implementing them

## HOW TO COMPILE
                    -PASTE THE COMMAND ON POWERSHELL ON THE PATH WHERE SOURCE FILES OF PROJECT ARE SAVED.
                    "g++ draw.cpp files.cpp game.cpp input.cpp logic.cpp main.cpp menu.cpp settings.cpp -I ../header -o ../build/SnakeGame.exe"

## CONTROLS 
            PRESS :
                W - Move Up
                A - Move Left
                S - Move Down
                D - Move Right

## PROJECT STRUCTURE 
                    SnakeGameProject/
                    |--src/-(Contains all .cpp files )
                    |--header/-(Contains all .h files)
                    |-- data/-(Contains all .txt & .data[binary files] files)
                    |-- build/-(Contains .exe file)
